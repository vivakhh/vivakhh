# 🚀 구글 전면 광고(Vignette) 기술 가이드

전면 광고는 일반 배너보다 클릭률(CTR)과 단가가 훨씬 높습니다.

## 1. 전면 광고 노출 조건
- **링크 클릭 필수:** 방문자가 <a href="..."> 링크를 클릭해 다른 페이지로 이동할 때만 뜹니다.
- **체류 시간:** 최소 10~15초 이상 글을 읽어야 구글이 광고를 띄워줍니다.
- **빈도 제한:** 한 사람당 1시간에 1번 정도만 노출됩니다. 첫 번째 클릭이 가장 중요합니다.

## 2. 티스토리 최적화 세팅
- **모바일 웹 자동 연결 해제:** 관리자 -> 모바일 -> '모바일 웹 자동 연결'을 **사용하지 않습니다**로 설정.
- **전면 광고 간격:** 애드센스 관리 -> 광고 -> 수정 -> 전면 광고 빈도를 최대로 설정.

## 3. 황금 버튼 활용
제가 만들어 드린 '액션 버튼'을 본문 상단(서론 끝)에 배치하여 독자가 가장 집중할 때 첫 번째 전면 광고를 노출하세요.
